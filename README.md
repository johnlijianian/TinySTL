TinySTL
=======
采用C++11实现一款简易的STL标准库，对于MySTL进行重构，去除冗余的设计（裁剪容器和算）。

目的：练习数据结构与算法和C++ Template编程

参考：
1. 《STL源码剖析》
2. [MySTL](https://github.com/Alinshans/MyTinySTL)
3. [TinySTL](https://github.com/zouxiaohang/TinySTL)
